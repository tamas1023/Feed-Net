.env file lérehozása az API könyvtárban
tartalma:
DB_HOST="localhost"
DB_USER="root"
DB_PASS=""
DB_NAME="Feed-Net"
port=8080
npm i az API könyvtárban
az adatbázis létrehozása
a letöltött file okat az xampp htdocs mappába tenni
